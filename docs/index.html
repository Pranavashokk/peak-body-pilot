// CAPABILITIES

λ

EWMA fatigue model

Acute (τ=4d) vs chronic (τ=28d) impulse decay — the Banister method applied to RPE-weighted tonnage.

Δ

Bodyweight regression

7-day linear trend on daily BW; aggressive drops trigger macro-adjustment warnings to protect lean mass.

∑

Per-muscle volume

Tracks weekly working sets per muscle group against MEV / MAV / MRV landmarks.

!

Deload detection

Sustained readiness deficit + rising RPE-at-load auto-flags a deload week before regression hits.

★

PR windows

Positive readiness + freshness on a target muscle opens a "PR_ATTEMPT" window in the planner.

⇄

Adaptive split

Next-session prescriptions per muscle: PUSH, MAINTAIN, RECOVER, or PR_ATTEMPT.
